wip

.x* introduction

this type of file is used in 3 known games:
Hot Wheels Velocity X
Snoopy vs. the Red Baron
Pac Man World Rally

the file extension changes based on console
pc: .xpc
gamecube: .xgc
xbox: .xdx
psp .xpp
ps2 .XPS

the file can contain a wide verity of data
sounds/music
geometry
textures
animations
ect

there are a number of differences between each game, console and file type
currently focusing on the gamecube version of pac man world rally,

Gamecube uses big endian

Offset Type Description
